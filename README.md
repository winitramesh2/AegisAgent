# Aegis Agent

AI-powered L1 support bot for authenticator Client Apps (Android, iOS, Desktop). It provides cloud-first diagnosis with local cross-verification, log analysis via OpenSearch, and guided troubleshooting with explicit escalation to email + JIRA when needed. Hybrid AI adds on-device inference and response packs for offline capability and lower latency.
